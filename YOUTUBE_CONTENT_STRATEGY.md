## 🎬 3. コンテンツ構造

### **3-1. ブログ記事（起点）**
- **役割**: 最も詳細な情報源（7,000-10,000文字）
- **内容**: 技術の第一原理、実装手順、コード例、ケーススタディ
- **SEO**: ベクトルリンク最適化済み（Fragment ID + Complete URI）

### **3-2. ショート動画（30秒・縦型）**
- **役割**: 入り口・バズ・エンゲージメント
- **構造**: 4フェーズ（Hook・Empathy・Body・CTA）
- **文字数**: 450-500文字
- **トーン**: エンジニア視点を保ちつつ、誰でもわかる言葉
- **目的**: SNS最大化、ブログへの導線

**配信先**:
| プラットフォーム | 形式 | 特徴 |
|-----------------|------|------|
| Instagram | **縦動画（リール）** | ビジュアル・人・ビジョン・文化 |
| TikTok | **縦動画（リール）** | トレンド・技術ミーム・エンタメ |
| YouTubeショート | **縦動画** | 駆動開発の思想と実装 |
| Lemon8 | **縦動画** | ライフスタイル・実用的Tips |

### **3-3. 中尺動画（2分・横型）**
- **役割**: 教育的価値・AI引用最適化・ブランド構築
- **構造**: 5フェーズ（Hook・Problem・Solution・Summary・CTA）
- **文字数**: 1,250-1,400文字
- **トーン**: プロフェッショナル、詳細解説
- **目的**: SEO・AI引用・Watch Time向上・専門性の証明

## 🎥 5. 動画コンテンツフォーマット

- **プラットフォーム**: Instagram、TikTok、YouTubeショート、Lemon8

- **構成**:
  - 0-5秒: Hook（問題提起）
  - 5-20秒: Problem（課題の詳細）
  - 20-90秒: Solution（5-7つの要点を詳細解説）
  - 90-110秒: Summary（まとめ）
  - 110-120秒: CTA（チャンネル登録、ブログ誘導）

## 🎯 8. コンテンツワークフロー（詳細）

### **Step 1: ブログ記事生成**
```
1. テーマ選定（駆動開発・ベクトルリンク・AI技術の本質）
2. RAGシステムで記事生成（7,000-10,000文字）
3. Fragment ID + Complete URI設定
4. ベクトルリンク化
5. nands.tech/posts/{slug} に公開
```

### **Step 2: ショート台本生成（30秒・縦型用）**
```
1. /admin/posts から「🎬 ショート台本生成」
2. OpenAI GPT-4o で台本生成
   ├─ ブログ記事の内容を忠実に要約
   ├─ **記事にある数字のみ使用**（ない場合は概念で補う）
   ├─ ペルソナ（Kento・Yumi）に合わせた視点
   ├─ チャンネルコンセプト（駆動開発・第一原理）を反映
   └─ SNS別最適化（6種類）
3. company_youtube_shorts テーブルに保存（Draft）
4. 台本確認 → 修正 → 承認
```

### **Step 3: 中尺台本生成（2分・横型用）**
```
1. /admin/posts から「🎯 中尺台本生成」
2. OpenAI GPT-4o で台本生成
   ├─ ブログ記事の詳細要約（1,250-1,400文字）
   ├─ 5-7つの要点を詳細に解説
   ├─ YouTubeチャプター（タイムスタンプ）生成
   └─ ペルソナに合わせた深度調整
3. company_youtube_shorts テーブルに保存（Draft）
4. 台本確認 → 修正 → 承認
```

### **Step 4: 動画編集**
```
【ショート動画（縦型・30秒）】
1. CapCut / Premiere Pro で編集
2. 9:16 比率（1080×1920）
3. テロップ大きめ、視覚的インパクト
4. BGM: テンポ速め
5. エクスポート: MP4、1080p

【中尺動画（横型・2分）】
1. Premiere Pro で編集
2. 16:9 比率（1920×1080）
3. チャプター設定（0:00、0:05、0:20...）
4. BGM: 落ち着いたテック系
5. エクスポート: MP4、1080p
```

### **Step 5: YouTube投稿**
```
【中尺動画投稿（19:30）】
1. YouTube Studio にアップロード
2. タイトル・説明文・タグをコピー（台本管理画面から）
3. チャプター設定（説明文に記載）
4. 再生リストに追加（4つのうち適切なもの）
5. 公開設定 → 公開

【ショート動画投稿（19:45）】
1. YouTube Studio にアップロード
2. タイトル・説明文・タグをコピー
3. 公開設定 → 公開
4. YouTube URL取得
5. /admin/youtube-scripts/{scriptId} でURL登録
   └─ ベクトルリンク化完了（AI引用対象に）
```

### **Step 6: SNS同時展開（20:15）**
```
【縦動画配信】
- Instagram: リール投稿 + キャプション（2,200文字）
- TikTok: 動画投稿 + キャプション（150文字）
- Lemon8: 動画投稿 + 投稿文（1,000文字）

【横動画配信】
- LinkedIn: 動画投稿 + タイトル（100文字）+ 本文（1,300文字）
- X: 動画投稿 + 投稿文（280文字）
- Threads: 動画投稿 + 投稿文（500文字）
```

### **Step 7: インデックス登録（20:00）**
```
1. Google Search Console にアクセス
2. URL検査ツールでブログURLを送信
3. インデックス登録をリクエスト
```

## 🔧 10. 現在のシステムの課題と改善点

### **❌ 現在の問題点**
1. **数字の正確性**:
   - 記事にない数字を生成している可能性がある
   - 例: 「工数50%削減」→ 記事に記載がない場合は使えない
