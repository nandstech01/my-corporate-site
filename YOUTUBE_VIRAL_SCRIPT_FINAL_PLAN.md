## 🎬 バズるテクニックの実装

### **ストーリー構造（But → So → But → So）**

| フェーズ | 内容 | 時間 | 目的 |
|---------|------|------|------|
| **Hook** | 問題提起（壁にぶつかった） | 0-2秒 | 注目獲得 |
| **But** | 隠そうとする（でも問題がある） | 3-5秒 | 共感 |
| **So** | 解決策を考える | 5-15秒 | 価値提供 |
| **But** | うまくいかない（新たな壁） | 15-20秒 | ツイスト |
| **So** | さらに工夫する（最終解決） | 20-28秒 | 感動 |
| **CTA** | 行動喚起 | 28-30秒 | 誘導 |

**例（ベクトルリンクの動画）**:
```
Hook（0-2秒）: 「ブログを書いても、AIに引用されない」
But（3-5秒）: 「SEO対策はしてるのに、なぜ？」
So（5-15秒）: 「実はAIは構造で理解する。Fragment化が必要」
But（15-20秒）: 「でも、普通のFragment化じゃダメだった」
So（20-28秒）: 「ベクトルリンクで意味単位にすると、AIが引用し始めた」
CTA（28-30秒）: 「詳しくはブログで。プロフィールから」
```

### **テンポ設計（5つの要素）**

| 要素 | 説明 | 実装 |
|------|------|------|
| **フック** | 衝撃的な一文 | 「これ知らないとヤバい」 |
| **フォアシャドウ** | 結末の伏線 | 「実は3つの理由がある」 |
| **スムーズな展開** | But→So→Butのリズム | テンポ速め、早口 |
| **クライマックス** | ツイスト（予想外の展開） | 「でも、それだけじゃない」 |
| **素早いクロージング** | 5秒でまとめ | 「詳しくはブログで」 |

### **ツイスト（予想外の展開）**

**Kenjiさん独自のツイスト**:
- 「でも、それは序章に過ぎない」
- 「実は、この設計思想には3層の構造がある」
- 「AIは文章を読むのではなく、構造をたどる」

**感情を動かすポイント**:
- **驚き**: 「え、そうなの？」
- **共感**: 「あるある、分かる」
- **解決**: 「なるほど、そういうことか」

## 🌐 SNS別最適化戦略（バズるために）

### **❶ X（Twitter）— 最重要プラットフォーム**

**現状の問題**: 反応がない → 薄っぺらい

**解決策**:
1. **冒頭で衝撃** - 「🚨これ知らないとヤバい」
2. **具体的な数字** - ブログ記事にある数字のみ
3. **損失回避** - 「知らないと損」「この実装ミス、致命的」
4. **1ツイート1メッセージ** - 複数のことを詰め込まない
5. **絵文字を効果的に** - 🚨⚡️💡🔥（多用しすぎない）

**構造**:
```
【冒頭1行】衝撃的な一文（損失回避型）
【本文2-3行】核心を凝縮（具体的な数字・事例）
【CTA】ブログ誘導 + ハッシュタグ2-3個

文字数: 250-280文字（ギリギリまで使う）
```

**良い例**:
```
🚨AIに引用されないブログ、9割が見落とす理由

SEO対策してもAIは構造で理解する。
Fragment化 + ベクトル埋め込みで
AIが引用し始めた実測データあり。

詳細👉 [ブログURL]
#ベクトルリンク #AI #エンジニア
```

**悪い例（避ける）**:
```
新しい記事を書きました。
ベクトルリンクについて解説しています。
ぜひ読んでください。

[ブログURL]
```

### **❷ LinkedIn — 専門性 × ビジネス価値**

**現状の問題**: 反応がない → プロフェッショナル感が薄い

**解決策**:
1. **タイトルで価値を明示** - 「〇〇でROI3倍」
2. **データドリブン** - 数字・事例・ケーススタディ
3. **ビジネス視点** - 技術 → ビジネス価値の流れ
4. **構造化された文章** - 箇条書き、番号付きリスト
5. **プロフェッショナルなトーン** - 落ち着いた語り口

**構造**:
```
【タイトル】ビジネス価値を要約（50文字）
【冒頭】課題の明確化
【本文】
  - 技術的解決策
  - ビジネスへの影響
  - 実装のポイント
【CTA】ブログ誘導・議論促進

文字数: 200-300文字
```

**良い例**:
```
【タイトル】
AIが引用できる構造設計でオーガニック流入3倍に

【本文】
AI検索エンジン時代、従来のSEOだけでは不十分です。

我々が実装した「ベクトルリンク設計」では：
✓ コンテンツをFragment化
✓ 意味単位でベクトル埋め込み
✓ AIが引用しやすい構造に

結果、オーガニック流入が3ヶ月で3倍に。
特にChatGPT、Perplexityからの流入が顕著です。

技術詳細はブログで公開中👉 [URL]

#AI #ビジネス戦略 #技術経営
```

```

【解決】どう解決したか
【学び】実務で使えるポイント
```

```
詳細👉 [ブログURL]
```
