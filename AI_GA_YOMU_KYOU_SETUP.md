# 「AIが読む今日」実装ガイド

## 🧭 コンセプト

**「AIが読む今日」- 本日のニュースを知りながらAIを勉強できるコンテンツ**

毎日のニュースを素材に、AIアーキテクトが「現実をベクトル的に読む」視点で再構成。
表層（出来事）→ 背景（意味）→ 深層（哲学）。

見るだけでAIの"理解構造"がわかる、
世界初の「AIリテラシー × ニュース解釈」コンテンツ。

---

## 🎯 目的

### **AIを技術ではなく、思考様式として学ぶ**

1. **現実のデータ入力層**: AIが今の世界をどう見るか = リアルタイムRAG
2. **思考の中層（構造化層）**: ニュースの裏を読む = 意味ベクトルの理解
3. **出力層（伝達形式）**: ブログ = 構造的ログ、ショート = 記憶に刺さるビジュアル表現

### **本質的な価値**

1日1ニュースで、AIが"世界をどう読むか"を体験できる。

ニュースサイトでもなければ、AI講座でもない。

**AIと共に世界を再解釈する体験型メディア。**

---

## 🏗 アーキテクチャ

```
1. /admin/trend-ragで今日のニュース登録
   ↓
2. ニュースをベクトル化してDBに保存（trend_rag）
   ↓
3. ショート動画台本生成時にトレンドRAG検索（必須）
   ↓
4. 関連度 > 0.4のニュースを取得
   ↓
5. GPT-5が「AI的再解釈」で台本生成
   - 「今日、〇〇が起きた。これ、AIで言うと"〇〇"です」
   - AI的再解釈 → 技術的概念 → ブログ記事のテーマ → 一言哲学
   ↓
6. Hookで使用（冒頭で注目を引く）
```

---

## 📝 台本構造

### **1️⃣ Hook（冒頭5秒・110文字以上）**
「今日、〇〇が起きた。これ、AIで言うと"〇〇"です」

**例:**
- 「今日、熊が街に出た。これ、AIで言うと"文脈の逸脱"です」
- 「今日、SNSで大炎上。これ、AIで言うと"感情ベクトルの連鎖"です」

### **2️⃣ Empathy（3-5秒・120文字以上）**
「AIを理解してないと、〇〇で損する」

**例:**
- 「AIを使ってる人、多い。でも、文脈を理解してない人も多い。突然エラー。なぜ？文脈が切れるから」

### **3️⃣ Body（10-15秒・180文字以上）**
AI的再解釈 → 技術的概念 → ブログ記事のテーマ → 一言哲学

**例:**
- 「文脈の逸脱、これが本質。熊は森から出た。AIも文脈から出る。だからLLMO。Large Language Model Optimization。つまり、文脈最適化。無駄な計算を減らす。ベクトルリンクで文脈を繋ぐ。一言哲学: "文脈を失うと、AIも人も迷う。"」

### **4️⃣ CTA（ラスト5秒・80文字以上）**
「AIが世界をどう読むか、ブログで解説中」

**例:**
- 「AIが世界をどう読むか、もっと知りたい方はブログで詳しく解説中。実装コード、構造化データも公開してます」

## 🚀 使い方

### **ステップ1: トレンドニュース登録**

```
http://localhost:3000/admin/trend-rag
```

1. 「サンプルニュース生成」をクリック（現在はサンプルのみ）
2. サンプルニュース（クマ、大谷翔平、桜など）が表示される
3. 「ベクトル化してDBに保存」をクリック

### **ステップ2: ショート動画台本生成**

```
http://localhost:3000/admin/content-generation
```

1. 記事を選択（例: LLMO記事）
2. 「ショート動画台本生成」をクリック
3. **自動的にトレンドRAG検索**が実行される
4. **トレンドRAGがない場合はエラー**（リアルタイムニュース必須）

**期待される結果（Hook）:**
```
「今日、熊が街に出た。これ、AIで言うと"文脈の逸脱"です。
熊は森（文脈）から街（未知）へ。AIも同じ。文脈を失うと迷う。
だからLLMO。文脈最適化の技術。」
```

## 🔧 カスタマイズ

### **1. サンプルニュースを変更**

`/app/admin/trend-rag/page.tsx`の`sampleTrends`を編集：

```typescript
const sampleTrends = [
  {
    title: '新しいニュースタイトル',
    content: 'ニュース本文',
    url: 'https://example.com/news/new',
    category: 'general',
  },
];
```

### **2. 関連度閾値を変更**

`/app/api/admin/generate-youtube-script/route.ts`の`threshold`を編集：

```typescript
const trendResults = await hybridSearch.search({
  query: postTitle,
  source: 'trend',
  limit: 3,
  threshold: 0.4, // ← ここを変更（0.0 - 1.0）
});
```
