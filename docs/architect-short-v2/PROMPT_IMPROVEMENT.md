## ✅ 改善内容

### 1. RAGフォーマットの大幅改善

```typescript
// 5件使用（3→5）
ragResults.deepResearch.results.slice(0, 5).forEach((r: any, i: number) => {
  
  // サマリー（全文）
  if (r.summary) {
    markdown += `**📝 要約:**\n${r.summary}\n\n`;
  }
  
  // 重要な発見（key_findings）✨ NEW!
  if (r.key_findings && r.key_findings.length > 0) {
    markdown += `**💡 重要な発見:**\n`;
    r.key_findings.slice(0, 5).forEach((finding: string) => {
      markdown += `- ${finding}\n`;
    });
    markdown += '\n';
  }
  
  // コンテンツ（1000文字）
  if (r.content) {
    markdown += `**📄 詳細:**\n${r.content.substring(0, 1000)}...\n\n`;
  }
  
  // ソースURL
  if (r.source_urls && r.source_urls.length > 0) {
    markdown += `**🔗 ソース:** ${r.source_urls.slice(0, 2).join(', ')}\n\n`;
  }
});
```

**改善点:**
- 3件 → 5件
- 300文字 → 1500文字以上/件
- `key_findings`を明示的に表示✨
- ソースURLも含める

---

### 2. 本体生成指示の強化

```markdown
### 3. 本体（20秒・約200文字）

⚠️ **重要:** 上記の「最新AI情報」から、
         最も衝撃的で具体的な情報を3つ選んで使用してください！

**えっ（驚き）:**
- 🧪ディープリサーチの「重要な発見」から最も驚くべき情報を選ぶ
- 具体的な数字、事例、統計データを必ず使う
- 「2025年、〇〇は△△兆円に！」のような具体性
- 専門用語は一切使わない
```

**強化点:**
- 「最も衝撃的な情報を3つ」と明示
- 「具体的な数字・統計データを必ず使う」と強制
- 例示を追加
