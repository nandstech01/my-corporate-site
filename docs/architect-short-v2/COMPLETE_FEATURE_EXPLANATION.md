## ❌ 現在の課題

### 1. 数字の選択

```
問題: GPT-4oが数字を丸めたり、抽象化したりする

例:
  RAG: 「6,858億7,300万円」
  台本: 「6,858億7,300万円」← 桁数が多すぎて分かりにくい

改善案:
  プロンプトに「読みやすい形式で表現」を追加
  「約6,900億円」「約2.5兆円」のような丸め方を許可
```

### 2. ストーリーの一貫性

```
問題: 市場規模 → 企業の投資 → 個人の年収 と話が飛ぶ

例:
  surprise: 市場規模（2.5兆円）
  reason: 企業の投資回収（12-18ヶ月）
  insight: 個人の年収（月85万円）

改善案:
  プロンプトに「一貫したストーリーラインを維持」を追加
  同じテーマ（例：個人の年収のみ）に絞る
```
