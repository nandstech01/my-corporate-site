## 📊 簡易度スコア計算

### 計算式

```typescript
simplicityScore = 
  (termPenalty + lengthScore) / 2

termPenalty = 1 - (専門用語数 × 0.1)
lengthScore = 1 - ((平均文字数 - 15) × 0.02)
```

### スコアの意味

```
0.9-1.0: 完璧（小学生でも理解可能）
0.7-0.9: 良好（中学生でも理解可能）
0.5-0.7: 改善必要（専門用語が多い）
0.0-0.5: NG（専門用語だらけ）
```
