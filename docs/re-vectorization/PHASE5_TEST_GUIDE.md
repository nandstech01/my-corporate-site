## 🐛 エラー発生時の対応

### エラーログ確認
```bash
# サーバーログを確認
# ターミナルで開発サーバーのログを確認

# ブラウザコンソールを確認
# F12 → Console タブ
```

### よくあるエラーと対処法

#### 1. 「記事IDが見つかりません」
**原因**: postIdが未設定
**対処**: ページをリロードして再度実行

#### 2. 「この機能はRAG記事（postsテーブル）のみ対応」
**原因**: ChatGPT記事で実行しようとした
**対処**: RAG記事で実行する

#### 3. 「再ベクトル化に失敗しました」
**原因**: APIエラー
**対処**: 
- サーバーログを確認
- OpenAI APIキーが設定されているか確認
- ネットワーク接続を確認

#### 4. タイムアウト
**原因**: Fragment ID数が多すぎる
**対処**:
- 処理完了を待つ（最大60秒）
- サーバーログでエラー確認
