## 📊 技術仕様

### アーキテクチャ

```
編集ページ (edit/page.tsx)
    ↓
    [🔄 再ベクトル化ボタン]
    ↓
既存API (/api/vectorize-blog-fragments)
    ↓
FragmentVectorizer クラス
    ↓
1. マークダウン解析
2. Fragment ID抽出
3. OpenAI Embeddings生成
4. fragment_vectors テーブル更新
```

### API統合

**エンドポイント**: `/api/vectorize-blog-fragments`  
**メソッド**: POST  
**パラメータ**: `{ postIds: [123] }`  
**処理時間**: 15-30秒（Fragment ID数に依存）  
**コスト**: 約$0.01-0.03/記事  

### データベース影響

**更新テーブル**: `fragment_vectors`  
**削除範囲**: 指定記事のFragment IDのみ  
**影響範囲**: 他の記事に影響なし  
**トランザクション**: 安全（再生成可能）
