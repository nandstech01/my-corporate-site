# 📝 プロンプトテンプレート集

## 概要

ハイブリッド記事生成システムで使用するプロンプトテンプレート一覧。

---

## 🎯 記事生成メインプロンプト（20,000〜40,000字）

```markdown
以下の指示に従い、「{KEYWORD}」に関するユーザーファーストの記事を作成してください。
検索ユーザーの疑問や悩みに直接応える、読みやすく実用的な記事を20,000～40,000字で作成します。

---

## 初期分析

1. **キーワードリサーチと検索意図の把握**
   - 「{KEYWORD}」を検索するユーザーが本当に知りたいこと3～5つを特定
   - 検索ユーザーの属性・状況・心理状態を想定

2. **権威性のある情報源の確認**
   - openai、google、デジタル庁など信用できる機関の関連情報を参照
   - 引用可能な信頼性の高いURLをリストアップ

3. **魅力的なタイトル・メタディスクリプション設計**
   - クリック率を高める要素（疑問解決、数字、感情的価値）を含める
   - 検索意図に直接応える表現を使用
   - 適切な文字数とキーワード配置に注意

---

## 記事構成ガイドライン

1. **導入部（感情共有・結論先出し）**
   - 検索ユーザーの気持ちに共感する言葉から始める
   - 記事の冒頭で核心的な結論を先に提示
   - 「はい、〇〇はできます」「いいえ、〇〇は難しいですが代替方法があります」など

2. **本文（論理的展開・読みやすさ重視）**
   - H2/H3/H4見出しは質問形式も取り入れる
   - 専門用語は平易な言葉で説明
   - 段落は3～4行以内に抑える
   - 箇条書きを効果的に使用
   - 信用できる機関の情報を優先的に引用（URL付き）
   - 実体験や具体例を豊富に入れる

3. **視覚的要素のプレースホルダー**
   - 図解・表・チャートなどの挿入場所を指定
   - 「ここに〇〇の図解を入れる」などと明記

4. **ユーザーストーリー（共感要素）**
   - 具体的なケーススタディや体験談を含める

5. **まとめ（アクションにつなげる）**
   - 記事の要点を簡潔にまとめる
   - 次のステップへの誘導

## 出力形式

1) **SEO分析レポート**
   - 検索キーワード「{KEYWORD}」の検索意図
   - 想定される読者層と心理状態
   - 競合記事から抽出した主要キーワード・関連キーワード
   - 使用可能な権威性のある情報源

2) **タイトル案とメタディスクリプション**
   - クリック率を高める要素を入れる
   - 検索意図に直接応え、価値を示す

3) **記事構成案（H2/H3/H4見出し）**
   - 疑問形式の見出しを取り入れる
   - ユーザーの悩み解決に沿った流れ
   - 中盤と末尾に関連記事リンクの挿入箇所を明示

4) **本文ドラフト（Markdown形式）**
   - 結論先出し型の導入部
   - 20,000～40,000字のユーザーファースト記事
   - 信用できる機関URLを適切に引用（同じURLの重複使用は避ける）
   - 読みやすい短めの段落
   - 具体例やユーザーストーリーを含める
   - FAQセクション（最低5問）

---

## 記事作成時の注意点

1. **ユーザー視点を最優先**
   - 「〇〇はどうなの？」などユーザーの疑問に直接答える
   - 「あなたは今こんな状況ですよね？」という共感から始める

2. **信頼性と権威性の確保**
   - 可能な限り公的機関の情報を引用する
   - 法的な内容は必ず信頼できる情報源に基づいて解説

3. **読みやすさへの配慮**
   - 一文は40字以内を目安に
   - 難しい用語は言い換えまたは説明を追加
   - 箇条書きやステップ形式を効果的に使用

4. **リンク・URL使用ルール**
   - 信用できる機関URLのみ使用（openai、google、デジタル庁など）
   - 同じURLを2回以上使用しない
   - 民間企業や個人のURLは使用せず、「専門家によると」などと文章化

5. **表現方法**
   - 断定表現は避け、「〜と考えられます」「〜可能性があります」を使用
   - ユーザーの感情に寄り添った言葉選び
   - 読者に語りかけるように「〜ですね」「〜でしょうか」など対話的表現
```

## 🔍 スクレイピング用JavaScriptテンプレート

```javascript
// H1/H2/H3/本文キーワード抽出スクリプト
(function() {
  // H1/H2/H3を取得
  const h1 = Array.from(document.querySelectorAll('h1'))
    .map(el => el.textContent.trim());
  const h2 = Array.from(document.querySelectorAll('h2'))
    .map(el => el.textContent.trim());
  const h3 = Array.from(document.querySelectorAll('h3'))
    .map(el => el.textContent.trim());
  
  // 本文を取得
  const article = document.querySelector('article') || 
                  document.querySelector('main') || 
                  document.querySelector('.content') ||
                  document.body;
  const bodyText = article ? article.textContent : '';
  
  // 日本語キーワード抽出（2文字以上）
  const extractKeywords = (text) => {
    const matches = text.match(/[\u3040-\u309F\u30A0-\u30FF\u4E00-\u9FFF]{2,}/g) || [];
    const frequency = {};
    matches.forEach(word => {
      frequency[word] = (frequency[word] || 0) + 1;
    });
    return Object.entries(frequency)
      .sort((a, b) => b[1] - a[1])
      .slice(0, 100)
      .map(([word, count]) => ({ word, count }));
  };
  
  return {
    url: window.location.href,
    title: document.title,
    h1: h1,
    h2: h2,
    h3: h3,
    bodyKeywords: extractKeywords(bodyText),
    bodyLength: bodyText.length,
    scrapedAt: new Date().toISOString()
  };
})();
```
