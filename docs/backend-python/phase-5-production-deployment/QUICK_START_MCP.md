## 🚀 Step 1: 事前準備（30分）

### 1-1: GCPプロジェクト作成（5分）

#### ブラウザで実施

**既存のGoogleアカウントを使用してOK！新規作成不要！**

1. https://console.cloud.google.com/ にアクセス
2. 右上の「プロジェクトを選択」→「新しいプロジェクト」
3. プロジェクト名: `rag-eval-prod`
4. 作成をクリック

#### 無料トライアル有効化

1. 左上の「≡」メニュー → 「お支払い」
2. 「無料トライアルを有効にする」
3. クレジットカード登録（$300クレジット付与、自動課金なし）

**✅ 完了**: GCPプロジェクト作成完了

---

### 1-2: Cloudflareアカウント作成（5分）

#### ブラウザで実施（MCP Browser使用可能）

1. https://dash.cloudflare.com/sign-up にアクセス
2. メールアドレス + パスワードで登録
3. メール認証

**⚠️ 重要**: ドメイン追加は後で実施（Step 4）

**✅ 完了**: Cloudflareアカウント作成完了

---

### 1-3: gcloud CLIインストール（10分）

#### macOSの場合

```bash
# Homebrewでインストール（推奨）
brew install --cask google-cloud-sdk

# インストール確認
gcloud --version

# 初期化
gcloud init

# ブラウザでGoogleアカウント認証
# プロジェクト選択: rag-eval-prod
# デフォルトリージョン: us-west1-b
```

#### 認証確認

```bash
# 認証状態確認
gcloud auth list

# プロジェクト確認
gcloud config get-value project
# 出力: rag-eval-prod
```

**✅ 完了**: gcloud CLI インストール完了

---

### 1-4: 既存ドメイン確認（10分）

#### 質問

**あなたの既存ドメインは何ですか？**

例:
- `mysite.com`
- `example.jp`
- `myblog.net`

→ **Phase 5では `rag.<あなたのドメイン>` を使用します**

#### ドメインがない場合

**オプションA: Cloudflare無料サブドメイン**（推奨）
- CloudflareでWorkers用の無料サブドメインを取得
- 例: `rag-eval.workers.dev`

**オプションB: 新規ドメイン購入**
- Cloudflare Registrarで購入（年$10程度）
- 完全無料ではなくなる

**オプションC: IPアドレス直接**
- SSL設定が面倒、非推奨

**✅ 確認完了**: ドメイン確認完了
