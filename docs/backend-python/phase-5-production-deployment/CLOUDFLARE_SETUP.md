## 🎯 プロキシステータスの意味

### 🟠 プロキシ オート（オレンジ雲）

- **トラフィックがCloudflareを経由**
- メリット:
  - CDN機能
  - DDoS保護
  - SSL/TLS自動設定
  - 実際のIPアドレスを隠蔽

### ⚪ DNSのみ（グレー雲）

- **トラフィックが直接サーバーに接続**
- 用途:
  - MXレコード（メール）
  - NSレコード（ネームサーバー）
  - TXTレコード（SPF, DKIM等）

### Step 2: お名前.comでネームサーバー変更

1. **お名前.com Navi**にログイン
   - https://navi.onamae.com/

2. ドメイン設定 → **ネームサーバーの変更**

3. 対象ドメイン: `nands.tech` を選択

4. 「他のネームサーバーを利用」を選択

5. Cloudflareの2つのネームサーバーを入力:
   ```
   ネームサーバー1: ns1.cloudflare.com
   ネームサーバー2: ns2.cloudflare.com
   ```

6. 「確認画面へ進む」→「設定する」
