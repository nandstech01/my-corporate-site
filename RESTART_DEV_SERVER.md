# 開発サーバー再起動手順

## ⚠️ 必須：開発サーバーを再起動してください

ハイブリッド検索の実装が完了しましたが、開発サーバーのキャッシュにより古いコードが使用されています。

### 再起動手順

1. **現在の開発サーバーを停止**
   ```bash
   # Ctrl+C で停止
   ```

2. **キャッシュをクリア**
   ```bash
   rm -rf .next
   ```

3. **開発サーバーを再起動**
   ```bash
   npm run dev
   ```

4. **ブラウザのキャッシュもクリア**
   - Chrome: Cmd+Shift+R（Mac）/ Ctrl+Shift+R（Windows）
   - または、DevToolsを開いて「Disable cache」にチェック

### 確認方法

再起動後、以下のコマンドでテストしてください：

```bash
curl -s http://localhost:3002/api/test-hybrid-search \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{"query":"ChatGPT","source":"company","limit":3}' | jq '.'
```

**期待される結果：**
- `totalResults: 3`
- BM25スコアとVectorスコアが両方取得できる
