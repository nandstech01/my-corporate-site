### 2. **価格設定の根拠**
   ```
   従来の工数:
   ├─ ブログ記事執筆: 3時間/本 × 月10本 = 30時間
   ├─ YouTube台本作成: 2時間/本 × 月10本 = 20時間
   ├─ 動画編集: 3時間/本 × 月10本 = 30時間
   ├─ SNS投稿文作成: 1時間/本 × 月10本 = 10時間
   └─ 合計: 90時間/月
   
   時給5,000円の場合: 450,000円/月
   
   👉 月額30万円〜50万円は**破格**
   ```

### 3. **デモの重要性**
   - 「百聞は一見に如かず」
   - 実際にあなたのブログ1本から台本生成 → HeyGen → Vrew → YouTube投稿
   - これを見せるだけで、商品価値は伝わる

   - ユーザーがブログURLを入力 → 台本を自動生成してメールで送信
