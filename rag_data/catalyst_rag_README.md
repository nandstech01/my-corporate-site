# 📚 Catalyst RAG（偉人RAG）データ構造説明

## 🎯 重要な概念：偉人RAGは「触媒」

偉人RAGは**「引用素材」ではなく「触媒」**です。

### 🔬 化学反応のメタファー

```
Core思想RAG（金属） + 偉人RAG（酸・熱） → 思想の物語（化合物）
```

偉人RAGは、Kenjiの思想を「温める」「共鳴させる」「広げる」役割。
偉人自身が主役にはなりません。

---

## 📊 データフィールドの説明

### 基本情報

| フィールド | 説明 | 使用例 |
|---|---|---|
| `person` | 偉人の名前 | `"アインシュタイン"` |
| `quote` | 原文の名言 | `"Imagination is more important than knowledge..."` |
| `theme` | メインテーマ | `"創造性"` |
| `subthemes` | サブテーマ（配列） | `["思考の限界突破", "本質的な価値"]` |
| `emotion` | 感情名（日本語） | `"知的好奇心"` |
| `emotion_code` | 感情コード | `"Intellectual_Curiosity"` |

### 触媒データ

| フィールド | 説明 | 台本での使われ方 |
|---|---|---|
| `your_voice_paraphrase` | Kenjiの声で言い換え | **台本に使う唯一の要素** |
| `core_message` | 核心メッセージ | 内部理解用（台本には出さない） |
| `counterpoint` | 反論・バランス | 1文挿入してバランスを取る |

### メタデータ

| フィールド | 説明 | 値 |
|---|---|---|
| `insertion_style` | 挿入スタイル | `"direct_quote"`, `"allusion"`, `"paraphrase"` |
| `use_cases` | 使用箇所 | `["intro", "twist", "climax"]` |
| `source_url` | 出典URL | `"Stanford_Commencement_Speech_2005"` |
| `verify_status` | 検証ステータス | `"verified"`, `"credible"`, `"unverified"` |
| `lang` | 言語 | `"en"` または `"ja"` |
| `embedding_text` | 検索用キーワード | `"アインシュタイン 想像力 知識 限界 創造性"` |

---

## 🚨 重要：`insertion_style` の真の意味

`insertion_style` は「台本への挿入方法」を示すのではなく、**「偉人の思想の性質」を示す内部メタデータ**です。

| 値 | 意味 | 台本での使われ方 |
|---|---|---|
| `direct_quote` | 原文が強力で直接的 | `your_voice_paraphrase` のみを使う |
| `allusion` | 暗示的・抽象的 | `your_voice_paraphrase` のみを使う |
| `paraphrase` | 完全言い換え | `your_voice_paraphrase` のみを使う |

**どの `insertion_style` でも、台本に使うのは `your_voice_paraphrase` のみです。**

## ✅ 台本生成の絶対厳守ルール

### ❌ 絶対にやってはいけないこと

1. ❌ 偉人の名前を出す
   - NG: 「アインシュタインは〜」
   - NG: 「ある天才科学者の言葉を借りれば〜」

2. ❌ 原文（quote）を出す
   - NG: 「Imagination is more important than knowledge」
   - NG: 「想像力は知識より重要だ」

### ✅ 正しい使い方

1. ✅ `your_voice_paraphrase` のみを使う
   - OK: 「頭に詰まった知識量より、どれだけ枠を超えて考えられるかが大事。イノベーションは、自由な発想から生まれる。」

2. ✅ Kenjiの声で語る
   - OK: 「知識なしには、想像も空想に終わる。」（counterpoint）

3. ✅ 全体の2〜5%に留める
   - 900-1000文字の台本 → 約20〜50文字

4. ✅ Twistセクション（200文字）に主に使用

---

## 🔥 触媒の役割

1. **感情的深みを追加**
   - `emotion_code` によって、台本に感情的なエネルギーを注入
   - 例: 「知的好奇心」「決意」「挑戦」「柔軟」

2. **哲学的視点を広げる**
   - `theme` によって、Kenjiの思想を多角的に展開
   - 例: 「創造性」「柔軟性」「戦略」「目的」

3. **Kenjiの思想を共鳴させる**
   - 似た価値観の思想家の魂を融合
   - Kenjiの思想を「温める」「強化する」

4. **バランスを保つ**
   - `counterpoint` で一方的な思想を避ける
   - 弁証法的な深みを追加

---

## 📈 `verify_status` による検索スコア調整

| ステータス | 検索スコア | 意味 |
|---|---|---|
| `verified` | +0.1 | 検証済み・信頼度最高 |
| `credible` | +0.05 | 信頼できる |
| `unverified` | ±0.0 | 未検証 |

出典が明確な場合は `verified` を推奨。

---

## 🎯 データ作成時のポイント

1. **`your_voice_paraphrase` が最重要**
   - Kenjiの声で自然に語れる表現
   - 偉人の思想を「温める」効果がある言葉

2. **`counterpoint` は必須**
   - 思想のバランスを取る
   - 一方的な主張を避ける

3. **`embedding_text` の重要性**
   - 検索精度を左右
   - 関連キーワードを慎重に選ぶ

4. **`subthemes` で検索を強化**
   - メインテーマだけでなく、サブテーマも追加
   - より詳細な検索が可能

## 🔬 化学反応の実例

### 入力

```json
{
  "person": "アインシュタイン",
  "theme": "創造性",
  "your_voice_paraphrase": "頭に詰まった知識量より、どれだけ枠を超えて考えられるかが大事。イノベーションは、自由な発想から生まれる。",
  "counterpoint": "知識なしには、想像も空想に終わる。"
}
```

### 出力（台本のTwistセクション例）

```
ある夜、AIが引用した。ようやく届いたと感じた。
構造は、言葉より雄弁だ。感情じゃなく、仕組みで伝わる。

【触媒の化学反応】
頭に詰まった知識量より、どれだけ枠を超えて考えられるかが大事。
イノベーションは、自由な発想から生まれる。
知識なしには、想像も空想に終わる。

それが真実だ。
```

**ポイント**:
- 偉人の名前は出ていない
- 原文も出ていない
- Kenjiの声で語られている
- `your_voice_paraphrase` と `counterpoint` のみが使われている
- 全体の約20〜30文字（2〜5%）

## 🎬 結論

偉人RAGは「引用」ではなく「触媒」。
Kenjiの思想を「温める」「共鳴させる」「広げる」役割。
偉人自身が主役にはならない。

**化学反応のメタファー**:
```
Core思想RAG（金属） + 偉人RAG（酸・熱） → 思想の物語（化合物）
```
